## خطة تحسين التصميم الشاملة

### المرحلة 1: تحسين صفحة تفاصيل الدورة (`/courses/[slug]`)

#### 1.1 Hero Section
- **الحالي**: تصميم بسيط مع خلفية secondary فقط
- **المطلوب**:
  - إضافة gradient overlay أنيق
  - تحسين تنسيق الـ tags (المستوى، المدة، عدد الدروس)
  - إضافة breadcrumb navigation
  - تحسين عرض صورة الغلاف مع rounded corners وshadow

#### 1.2 بطاقة التسجيل (CourseEnrollCard)
- **الحالي**: بطاقة بسيطة مع sticky positioning
- **المطلوب**:
  - إضافة تأثير shimmer على زر "سجل الآن"
  - إضافة أيقونة ضمان (money-back guarantee)
  - تحسين عرض السعر مع تأثير بصري أقوى
  - إضافة countdown timer اختياري للعروض

#### 1.3 تبويبات المحتوى
- **الحالي**: تبويبات أساسية
- **المطلوب**:
  - تحسين تصميم الـ tabs مع underline animation
  - إضافة أيقونات للتبويبات
  - تحسين عرض المنهج (Curriculum) مع accordion أنيق

### المرحلة 2: تحسين مشغل الفيديو (`/courses/[slug]/watch`)

#### 2.1 Layout
- **الحالي**: تصميم function-first
- **المطلوب**:
  - تحسين الـ sidebar مع progress bars لكل درس
  - إضافة mini-player عند الـ scroll
  - تحسين ألوان الـ dark theme

#### 2.2 Video Player
- **الحالي**: مشغل مخصص يعمل جيداً وظيفياً
- **المطلوب**:
  - تحسين تصميم الـ controls bar
  - إضافة chapter markers (اختياري)
  - تحسين الـ progress bar مع hover preview

### المرحلة 3: تحسين صفحة التسجيل (`/courses/[slug]/enroll`)

#### 3.1 نموذج التسجيل
- **الحالي**: نموذج واضح مع التحقق من الهاتف
- **المطلوب**:
  - إضافة steps indicator (خطوة 1: البيانات → خطوة 2: التحقق → خطوة 3: الدفع)
  - تحسين عرض ملخص الدورة
  - تأثيرات حركية عند الانتقال بين الخطوات

### المرحلة 4: تحسين بطاقة الدورة في الصفحة الرئيسية

#### 4.1 CourseServiceCard
- **الحالي**: بطاقة 3D مع تأثيرات gyroscope
- **المطلوب**:
  - تحسين عرض المعلومات (السعر، المدة، عدد المشتركين)
  - إضافة hover overlay مع "عرض التفاصيل"
  - تحسين التوافق مع الموبايل (تبسيط التأثيرات 3D)

### المرحلة 5: تحسين لوحة إدارة الدورات

#### 5.1 قائمة الدورات (`/admin/courses`)
- **الحالي**: قائمة بسيطة
- **المطلوب**:
  - إضافة إحصائيات سريعة (عدد الدورات، إجمالي المشتركين)
  - تحسين بطاقات الدورات مع المزيد من المعلومات
  - إضافة bulk actions (نشر/إيقاف متعدد)

#### 5.2 محرر الدورات (`/admin/courses/new`)
- **الحالي**: نموذج طويل واحد
- **المطلوب**:
  - تقسيم المحرر إلى خطوات (مع Tab navigation في الأعلى)
  - إضافة معاينة مباشرة (Live Preview panel)
  - تحسين تجربة الـ drag-and-drop للأقسام والدروس
  - إضافة auto-save draft feature

## معايير التصميم العامة

### الألوان
```
Primary:    #edbf8c (ذهبي فاتح)
Secondary:  #10302b (أخضر داكن)
Accent:     #ead3b9 (بيج فاتح)
Background: #fdf6ee (أبيض مائل للبيج)
```

### الخطوط
- **عربي**: Dubai (font-dubai)
- **إنجليزي**: Bristone (font-bristone)

### الزوايا
- بطاقات: `rounded-2xl` (20px)
- أزرار: `rounded-xl` (16px)
- inputs: `rounded-xl` (16px)
- badges: `rounded-full` أو `rounded-lg`

### الظلال
- بطاقات عادية: `shadow-lg`
- بطاقات hover: `shadow-2xl`
- عناصر عائمة: `shadow-xl`

### الحركات (Framer Motion)
- استخدام `fadeInUp` للعناصر الداخلة
- استخدام `staggerContainer` للقوائم
- استخدام `whileInView` بدل useScrollAnimation للأداء الأفضل
- مدة الحركة: 0.3-0.5 ثانية
- سهولة الحركة: `[0.4, 0, 0.2, 1]`
